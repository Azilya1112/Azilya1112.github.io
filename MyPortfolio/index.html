Разработка автоматизированной информационной системы "Рекламное агентство "Frischee Idee"

Общая информация:
Рекламное агентство "Frischee Idee" это проект, ориентированный на предоставление полного спектра услуг в области рекламы и маркетинга. Управляющим предоставляется возможность эффективного управления всеми аспектами рекламных кампаний, включая создание, запуск и мониторинг. Для клиентов, заказывающих рекламу, предоставляется удобная платформа для размещения заказов, отслеживания статуса рекламных кампаний и взаимодействия с командой агентства. Рекламное агентство стремится обеспечить высокое качество услуг, уникальный дизайн и эффективные рекламные решения для удовлетворения потребностей клиентов.

Разработка программы для отслеживание работ в цехе по обслуживанию техники на птицефабрике “Агро-Куш”

Общая информация:
Современные производственные предприятия, включая птицефабрики, требуют эффективного управления оборудованием и техническим обслуживанием для обеспечения бесперебойной работы и повышения производительности. Настоящее техническое задание направлено на разработку программы для цеха обслуживания техники на птицефабрике. Цель программы — оптимизация процессов управления работами, контроль за состоянием оборудования и эффективное техническое обслуживание. Программа предоставит персоналу инструменты для систематического планирования, отслеживания и анализа проведенных работ, способствуя повышению производительности и сокращению времени простоя оборудования.
Технологии:
Язык программирования: Java
Фреймворк: SpringBoot
База данных: Postgresql
